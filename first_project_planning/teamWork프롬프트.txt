이 이미지들은 내가 만드려고 하는 업무협업 프로그램의 레퍼런스야. 이미지 제목에 숫자와 단어들이 있는데, 번호는 1부터 13까지 이 사이트의 기능 구현 순서이고, 각 이미지의 제목은 그 이미지가 보여주는 화면의 기능 이름이야. 예를 들어, 첫번째 이미지인 1.main.png 는, 사이트에 처음 접속하면 보여지는 메인화면이야. 이후 2.mail.png 는, 1.main.png에 보이는 메인화면에서 좌측 상단부터 시작해서 첫 번째 기능인 메일 요소를 클릭하면 보여지는 이미지야. 이런식으로 그 다음 이미지에서 옆에 있는 요소들을 클릭하면 그 다음 이미지가 나오는 형식이야. 각각의 기능들이 러프하나마 구현 된 웹 사이트들을 만드려고 해. 아직 기획 단계니까 실제 데이터들을 입력하고 db에 연동할 것 까진 아니고, 그냥 표현만 됐으면 좋겠어. 기반 코드는 java ee 로 구현할거야. 아무래도 프로그램 완성본을 한 방에 만드는 건 조금 힘드니까, 간단하게 html 형식으로 디자인만 잘 깔끔하게 녹여서 구현됐으면 좋겠어. 이미지에 써져있는 김혜림이라는 이름은 전부 다 김태호로 바꿔줘. 그리고 실제 백엔드에 기술구현까지 전부 다 된 코드를 원하는 건 아니고,(그건 너무 규모가 커지니까) 기획단계에서 이런걸 만드려고 한다 정도로 실제 사이트가 동작은 하도록. 각각의 요소들을 누르면 실제로 페이지가 넘어가고 뭔가가 보이기는 하는 정도. 실제 기능구현이 되는 것처럼 html css javascript를 이용해서 프론트엔드 위주로 만들어줬으면 좋겠고, 만약 추가로 너가 제안하거나 더 개발이 가능하다거나, 실제 프로그램을 그냥 이 자리에서 만들어줄 수 있다거나 하면 (좋은 퀄리티로 가능하다면) 그렇게 해줘도 좋아.
코드는 최대한 깔끔하게 주석을 달아서 설명을 해줘야하고, 바로 만드는 게 아니라 이 프롬프트 이후에 너와 내가 좀 더 대화를 통해 무엇을 만들지 구체적으로 좁혀나가는 것도 좋아. 대충 보이는 결과물을 원하는게 아니라 정교하고도 내 의도가 명확히 담긴 훌륭한 결과물을 원해.
